## 配置说明
- 20210928添加了代理
```shell
set http_proxy=http://127.0.0.1:1081
```

- cfg下需要配置coinmarket.com的apikey，可以去申请[coinmarketcap.com/api](https://coinmarketcap.com/api/)

- 操作配置文件在holds目录下，json格式，多条记录分开
```json
{
  "name": "btc",
  "records": [
    {
     "operate": "+",
     "amount": 0.01329424,
     "sum": 613.82
    },
    {
     "operate": "-",
     "amount": 0.01329424,
     "sum": 613.82
    }
  ]
}
```
- name 币种
- records 表示操作记录
  - opetare 操作 “+”买入   “-”卖出
  - amount 数量
  - sum 总金额 单位usdt

**一个币种一个json文件**
